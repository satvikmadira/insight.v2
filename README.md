# 📊 InsightAI — BI Dashboard

AI-powered dashboard. Ask questions in plain English, get instant charts.

## How to Run

1. Download the code — click green Code button → Download ZIP
2. Extract the ZIP
3. Double click start.bat (Windows)
4. Browser opens at http://localhost:8000

## Add AI Key
1. Go to Settings in the app
2. Select Groq as provider
3. Get free key at console.groq.com/keys
4. Paste key and click Save

## Tech Stack
- Frontend: HTML CSS JavaScript
- Backend: Python FastAPI
- Database: SQLite
- AI: Groq Llama 3.3 (Free)
